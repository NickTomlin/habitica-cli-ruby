# habitica-cli

A command line interface for habitica

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'habitica-cli'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install habitica-cli

## Usage

```shell
habitica <command> <action>
```
